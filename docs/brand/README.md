# Allure Station — Brand assets

The mark pairs an **ascending report** (history & trends) with a **live broadcast node** — results
pushed from any pipeline, served back to the whole team in real time. Wordmark set in **Sora**,
shipped as vector outlines (no font files needed to render the logo).

## Files

| File | Use |
|---|---|
| [`allure-station-icon.svg`](allure-station-icon.svg) | Primary app icon (gradient). Also the web favicon (`packages/web/public/favicon.svg`) and header mark. |
| [`allure-station-lockup.svg`](allure-station-lockup.svg) | Horizontal logo for **light** backgrounds (dark wordmark, transparent). |
| [`allure-station-lockup-dark.svg`](allure-station-lockup-dark.svg) | Horizontal logo for **dark** backgrounds (ink panel, white wordmark). |
| [`allure-station-mark-mono.svg`](allure-station-mark-mono.svg) | 1-color mark using `currentColor` — inverts for dark mode / stamps on any context. |
| [`allure-station-icon-hub.svg`](allure-station-icon-hub.svg) | Alternate "hub" icon (one station wired to many projects) — infrastructure-forward read. |

The README hero uses the two lockups via a `<picture>` element so it adapts to GitHub's light/dark themes.
