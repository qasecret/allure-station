# README screenshots

Drop UI screenshots here, then uncomment the image block in the root
[`README.md`](../../README.md#screenshots).

Suggested set (PNG, ~1400px wide, light theme unless noted):

| File | Shows |
|---|---|
| `projects.png` | Project list (search + pagination). |
| `report.png` | A run's embedded Allure report. |
| `trends.png` | Trend dashboard (pass-rate / flakiness / duration). |
| `compare.png` | Run comparison (newly failing / fixed / flaky). |

Optional: `dark.png` (dark theme), `login.png` (local + "Sign in with SSO").

Tip: capture against the [quick start](../../README.md#quick-start) instance at
`http://localhost:5050` after pushing a couple of runs so the trend/compare views
have data.
